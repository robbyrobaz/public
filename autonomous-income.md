## AI Support Layer — Scoring Note
For any customer-facing SaaS idea, scores assume an AI support layer:
Claude API handles 85% of tickets autonomously. Remaining 15% auto-create
GitHub issues (tagged: bug / feature-request / question) for async review.
Human time: ~2–4 hrs/month regardless of customer count. This bumps passivity
+2 points vs. human-support baseline. See [AI-SUPPORT-PATTERN.md](./AI-SUPPORT-PATTERN.md).
